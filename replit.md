# Warung Sate Mbah Biyong

## Overview
A static HTML website for "Warung Sate Mbah Biyong", a traditional Javanese sate restaurant established since 1982. The site showcases the restaurant's menu, allows ordering, and provides contact information.

## Project Architecture
- **Type**: Static HTML website (no build step, no backend)
- **Files**:
  - `index.html` - Main landing page with menu, hero section, and navigation
  - `pesan.html` - Order/message page
- **Styling**: Tailwind CSS via CDN, Google Fonts (Plus Jakarta Sans, Material Symbols)
- **Deployment**: Static file hosting

## Development
- Served via Python's built-in HTTP server on port 5000
- No build process required
